# Instagram Clone

Small Instagram-style frontend with a focus on HTML and CSS. The goal of the project is to build the layout and content as cleanly as possible with semantic HTML instead of relying on lots of neutral `div` containers.

As a stretch goal, `index.js` was also provided. It stores the posts in an array and renders them dynamically into the DOM. This keeps the content separate from the markup and makes it easier to extend.

## What This Project Is About

- focus on modern CSS layout
- more semantic HTML
- dynamic rendering of posts with JavaScript

## Stretch Goal

`index.js` renders the post data from an array. That makes the page feel closer to a real data-driven UI than static HTML.
